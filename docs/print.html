CQUPT Piper

CQUPT Piper

CQUPT Piper 是一个用 Python 编写, 为 CQUPTer 提供自动登录教务在线查询信息服务的命令行小工具.

当前支持的查询信息有:

  • 学分
  • 学费
  • 绩点排名
  • 考试安排
  • 学生照片

该工具始于作者懒得每次都要打开浏览器输入验证码登录教务在线查询信息

该工具会在第一次使用时, 要求您输入学号和教务在线的登录密码, 仅仅是为了通过教务在线的身份验证和防止非本校的人使用. 除了您自己的电脑上, 系统不会在任何地方保存您的密码.

CQUPT Piper 官方交流QQ群: 972606439

安装

安装前, 确保您的设备已经安装了 Python3+ 和 pip

pip install cqupt

安装过程中可能会中断, 多试几次就好. 或者使用将pip源设置为阿里云镜像)

使用

Window 的同学, 打开 CMD 或者 PowerShell (不知道怎么打开?)

MacOS 和 Linux 的同学随你怎么搞

查询学费

cqupt --fee 2019

该命令会查询并显示 2019-2020 学年您的学费缴纳记录.

若不输入年份, 则会查询和显示大一到当前学年的学费缴纳记录

查询学生照片

cqupt --photo 2017213056

该命令会查询和显示学号为 2017213056 的同学的照片, 慎用哦 :)

若不输入学号, 则会查询和显示自己的照片.


执行下面命令, 可查看更多使用方法

cqupt -h

学费查询

命令

cqupt --fee <年份>

示例

查询 2018-2019 学年的学费缴纳情况

cqupt --fee 2018

查询大一到当前学年的学费缴纳情况

cqupt --fee

学分查询

命令

cqupt --credit <年份>

示例

查询 2018-2019 学年的学分数据

cqupt --credit 2018

查询大一到当前学年的学分数据

cqupt --credit

绩点排名查询

命令

cqupt --gpa <年份>

示例

查询 2018-2019 学年的绩点和年级排名情况

cqupt --gpa 2018

查询大一到当前学年的绩点和年级排名情况

cqupt --gpa

考试安排查询

命令

cqupt --task

示例

查询当前学期的考试安排

cqupt --task

由于考试安排数据教务在线更新缓慢, 查询结果可是仍然是上学期的数据

学生照片查询

命令

cqupt --photo <学号>

示例

查询学号为 2017213056 的同学教务在线上存的照片

cqupt --photo 2017213056

查询自己教务在线上存的照片

cqupt --photo

其他命令

更改绑定

cqupt --auth

更改绑定的学号或者更改当前绑定学号的密码可执行上面命令, 然后按照提示输入即可

⚠️⚠️ 这样更改的密码只是更改了 CQUPT Piper 里记录的密码, 并不会影响您教务在线的密码.

显示绑定的信息

cqupt --whoami

顾名思义, 该命令用来显示已绑定的学号和密码

清理缓存

cqupt --clear

该命令用来删除保存的上次登录的 Cookie, 使得下次使用需要重新请求登录

删除配置

cqupt --drop

该命令用来删除本地 CQUPT Piper 的所有配置文件和绑定的学号, 使得下次使用会要求重新绑定学号
