CQUPTPiper

CQUPT Piper

这个工具来源于作者懒得每次想去教务在线查信息时都要打开浏览器, 输入网址, 输入那智障验证码, 找半天才找到要查的信息在哪个地方 :)

目前支持查询的信息有

  • 学生照片
  • 学分
  • 学费
  • 绩点和排名
  • 考试安排

安装

安装前, 确保您的设备已经安装了 Python3+ 和 pip

使用 pip 安装

pip install CQUPTPiper

使用

Windows 的童鞋按 Win + R 输入 cmd 或者 Power Shell, 运行下面命令

piper

Linux 和 MacOS的童鞋, 随便您怎么搞.


运行 piper 后您会看到下面这样的语言选择提示

$ Keep on using English? [y/N]

如需使用简体中文, 就输入 N 然后回车

作为验证您是否是重庆邮电大学的在读学生, 第一次使用会要求输入教务在线的账号和密码. 另外, 作者发4, 如果保存了用户的密码, 作者1辈子找不到媳妇儿 [滑稽]

基本使用

登录成功后, 你会进入 piper 的 Shell 环境, 像下面这个亚子 (如果你使用的中文)

登录成功!
输入 help 查看可执行指令
>

所以, 输入 help 回车, 就能看到如何使用了.

切换语言

如果您吃饱了撑的, 想从中文换成英文, 您可以在命令行运行下面这个命令

piper -l en

换回中文当然就是

piper -l ch

目前 piper 只支持中英文, 以后也不会支持其他语言
